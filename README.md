# Python Mini Game

This project is a simple game application built using Python.
It includes two games: Stone-Paper-Scissors and Dice Roll.

## Features

* Play Stone-Paper-Scissors against computer
* Dice roll game with random outcomes
* Menu-driven program using `while True`
* Uses functions and separate logic file

## Project Structure

* `main.py` → Handles menu and user interaction
* `game_logic.py` → Contains game logic and random functions

## Technologies Used

* Python
* Random module
* Functions

## How to Run

1. Open the project folder
2. Run `main.py`
3. Choose a game from the menu

## Sample Gameplay

* Select option 1 → Play Stone-Paper-Scissors
* Select option 2 → Play Dice Game
* Select option 3 → Exit
